# Lyra3 Plugin (Google Lyria AI Music)

This plugin integrates with the Google Lyria AI music model (also referred to as Lyra3 in Dialtone).

## Features
- Generate music from text prompts.
- Control genre, era, instruments, and dynamics.
- Support for custom lyrics and vocal profiles.

## Versioning
- `src_v1`: Initial implementation using Google Lyria prompting standards.

## Usage
```sh
./dialtone.sh lyra3 src_v1 help
```
